# Practice Menu: Operation Drills (MUS-40H)

Operation is muscle memory and map memory. These drills build the reflexes that make the machine transparent: finding things fast, knowing what every knob does, never losing work. They are short on purpose. Do a few every day; rotate so each gets reps across the week.

## How to use this menu

- **Every session, before the lecture:** pick one **Navigation** drill (5 min). This is the warm-up that pays off most.
- **Mid-course (Week 2 on):** add one **Engine** drill.
- **Week 3:** add one **Connection** drill.
- Keep each drill **short and focused**. The goal is speed and certainty, not duration. Stop while it still feels easy; come back tomorrow.

## The rules that make drills work

1. **No notes, no peeking.** A drill you do with the manual open trains the wrong thing. If you cannot do it cold, that *is* the signal of what to practice.
2. **Read the screen out loud.** Saying what each knob and pad does right now is how the navigation logic sets in.
3. **Save before and after risky drills** (anything that copies, deletes, or reorganizes). Drills should never cost you real work.
4. **Time the slow ones loosely.** "Reach any setting in 10 seconds" is a real target; watch it drop over the weeks.
5. **End on a clean save.** Always.

## Navigation drills (every day)

**Name-the-control (2 min).** Point at five controls at random; say each one's job in under two seconds. The foundation of everything.

**Blind menu reach (3 min).** Name a setting (brightness, metronome, pad sensitivity, USB routing, MIDI clock); race to reach it from sample mode. Cut the time each day.

**Function-knob-pad reps (3 min).** Do the core gesture ten times on different functions (play mode, level, pitch, bus routing). Wear the groove in.

**Read-the-screen (2 min).** Enter three contexts (a pad's params, an effect, the record screen) and say what [VALUE] and [CTRL 1–3] do in each, confirming by turning them.

**Which-mode (1 min).** Jump between sample mode, [PATTERN SELECT], and DJ mode; each time, say out loud what a pad press does right now. Never be lost about which mode you are in.

## Pad & performance drills

**Eyes-closed pad-find (3 min).** Pick four pads; play them in patterns (1-2-3-4, then 1-3-2-4) with eyes closed. Stop looking at the pads.

**Mode-on-demand (2 min).** Name a play mode (gate / one-shot / loop); set a pad to it and demonstrate the difference in under five seconds.

**Hold-and-drum (3 min).** Hold a loop with [REMAIN], finger-drum a one-shot over it in time, eyes off the pads.

## Signal & recording drills

**Ten-second level set (3 min).** Arm an input on whatever is playing; set a healthy meter; disarm; repeat. Make it a reflex you never skip.

**Arm-source-level-go (3 min).** Run the first four record-flow steps (arm, source, level, dry) without recording, until automatic.

**Skip-back catch (2 min).** Play something, then grab the last seconds with skip-back without arming first. Trust the buffer.

**Clean-loop nudge (4 min).** Set any rhythmic sample to loop; nudge the end point until the seam vanishes. New sample each day.

## Engine drills (Week 2 on)

**Route-and-name (3 min).** Route a pad to a bus, turn on the bus effect, say "this affects only this pad" or "this affects everything." Always know what an effect touches.

**Name-the-category (2 min).** Call up any effect; before tweaking, say its category (character / tone / time / space / mix / performance) and a bus you would put it on.

**Loop-and-layer (4 min).** Record a two-part pattern live (kick, then snare) as it loops. Build the in-time recording reflex.

**Skeleton-then-feel (4 min).** Step-program a bare beat in TR-REC, then overdub hats live. Practice the handoff between the two methods.

**Print-it-down (3 min).** Take whatever you are working on and resample it to one pad, then continue. Make resampling automatic.

**Hit-the-limit (2 min).** Stack effects until the machine warns it cannot add more; then resample one to free the engine. Make the limit a known, solved thing.

## Connection drills (Week 3)

**Enter-and-assign (2 min).** Enter DJ mode, assign both channels, mix them with the [CTRL] knobs, exit. Make setup automatic.

**Interface check (2 min).** Connect the SP, confirm the computer sees it, set USB routing to MIX OUT, disconnect. Remove the friction to recording your work out.

**Leader-or-follower (2 min).** For any two-device setup, state which leads the clock and which follows, and how to set it.

**Round-trip (3 min).** Move one file onto the SP and one off it (app or card). Make import/export routine.

**Diagnose-out-loud (3 min).** Name a symptom from the Day 19 troubleshooting table; say the cause and fix before touching the machine.

---

## A sample 10-minute daily warm-up

1. Name-the-control (2 min)
2. Blind menu reach (3 min)
3. Eyes-closed pad-find (2 min)
4. Ten-second level set (3 min)

From Week 2, swap one in for Route-and-name or Print-it-down. From Week 3, add a Connection drill. Then start the day's lesson with the machine already warm and your hands already moving.
